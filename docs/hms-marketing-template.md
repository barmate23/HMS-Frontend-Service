# HMS Cloud Marketing Template

## Project Analysis

HMS Cloud is a modern hotel management system frontend built with Angular. The project presents itself as a complete hotel operations workspace for front office, reservations, room management, housekeeping, billing, laundry, inventory, purchase, POS, hotel setup, and user administration.

The product message should focus on one clear promise:

**Run hotel operations from one connected system, with live visibility across rooms, guests, billing, housekeeping, inventory, and restaurant/POS activity.**

### Strong Marketing Points

- Central dashboard for room inventory, occupancy, bookings, booking revenue, and POS performance.
- Front office command center with live room dashboard, floor-wise room map, booking status, guest details, arrivals, departures, and reservation workflow.
- Reservation tools including new booking, Gantt chart, arrivals, departures, and guest profiles.
- Housekeeping dashboard with room readiness, open tasks, maintenance blockers, SOP checks, staff load, lost and found, and room-level status.
- Billing module for guest folios, payments, invoices, refunds, inward stock, and vendor bills.
- Laundry workflow for order creation, tracking, item-level details, status flow, vendor management, and price master.
- Inventory and purchase modules for stock ledger, purchase requests, store issues, suppliers, purchase orders, and item configuration.
- POS module for outlets, table dining, room service orders, takeaway, billing, menu management, coupons, tax rules, and table actions.
- Hotel setup and master data modules for hotels, floors, room types, rooms, rate plans, departments, roles, shifts, and access audit.
- Secure staff login with password reset, first-time password setup, and role-based user management.
